Se añadio un modo de damas chinas que se juega contra la computadora se añadio una interfaz grafica para el terminal 


# Guia compilacion mac

1. Ejecutar este comando en la terminal
   
g++ -std=c++17 main.cpp juego.cpp menu.cpp ia.cpp movimientos.cpp validaciones.cpp -o damas

2. Ejecutar .exe con:
   
./damas


# Guia compilacion linux

1. Ejecutar este comando en la terminal
   
g++ -std=c++17 main.cpp juego.cpp menu.cpp ia.cpp movimientos.cpp validaciones.cpp -o damasç

2. Ejecutar .exe con:
./damas


# Guia compilacion windows


1. Abrir ruta de la carpeta con el comando cd
   
2. Ejecutar en la terminal

g++ -std=c++17 main.cpp juego.cpp menu.cpp ia.cpp movimientos.cpp validaciones.cpp -o damas.exe

3. Ejecutar  .exe con:
